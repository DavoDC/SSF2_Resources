# SSF2_Resources

[![ko-fi](https://ko-fi.com/img/githubbutton_sm.svg)](https://ko-fi.com/G2G31WKOCN)

A collection of resources for Super Smash Flash 2 - including scripts, configuration files, and more.

## Contents

- **scripts/** - `INSTALL_SSF2.sh` - installs SSF2 on Linux (Native, Wine Portable, Wine Installer)
- **files/** - desktop shortcut, Aussie tournament ruleset
- **tests/** - developer tests for the install script
- **docs/** - ideas and history for the install script

## Linux Install Script

If you're trying to install SSF2 on Linux, see [scripts/LINUX_INSTALL_GUIDE.md](scripts/LINUX_INSTALL_GUIDE.md).
