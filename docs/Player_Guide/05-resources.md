## 5.3 Character-Specific Resources {#5.3-character-specific-resources}

---

* Check the pins in the [character channels](https://discord.com/invite/mcleodgaming) if you want character-specific advice.  
* Find the [wiki page of the character](https://mcleodgaming.fandom.com/wiki/Category:Characters_in_Super_Smash_Flash_2) you are interested in, e.g. [Black Mage](https://mcleodgaming.fandom.com/wiki/Black_Mage_\(Super_Smash_Flash_2\)).  
* Search YouTube for that character to find guides and combo videos (e.g. [this search](https://www.youtube.com/results?search_query=captain+falcon+ssf2+)).  
* Check the [Character Resources document](https://docs.google.com/document/d/1Df4ULOweoKbBsFvNituJkoJG5ehCRd1PmC4au6z6Oqo/edit).
