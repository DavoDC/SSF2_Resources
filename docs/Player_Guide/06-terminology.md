# **6\. Terminology** {#6.-terminology}

---

| Term | Meaning |
| ----- | ----- |
| AI | “[Artificial intelligence](https://mcleodgaming.fandom.com/wiki/Artificial_intelligence)”. AKA the “Computer Player” or “CPU” (“Central Processing Unit”). |
| Bair  | “[Back Air/Aerial](https://mcleodgaming.fandom.com/wiki/Back_aerial)”. AKA “B-Air”. |
| BF | “[Battlefield](https://mcleodgaming.fandom.com/wiki/Battlefield_\(Super_Smash_Flash_2\))”, a stage. |
| Bo3 | “[Best of 3](https://en.wikipedia.org/wiki/Playoff_format#Best-of-three_playoff)”. Often done in the early sets of a tournament. |
| Bo5 | “[Best of 5](https://en.wikipedia.org/wiki/Playoff_format#Best-of-five_playoff)”. Often done in the quarterfinals, semifinals and finals of a tournament. |
| C-Stick | There are four C-Sticks on the [controls screen](#2.3.3-controls-settings) (C-UP, C-LEFT, C-DOWN, C-RIGHT). When activated while grounded, they perform uncharged [smash attacks](https://mcleodgaming.fandom.com/wiki/Smash_attack) in their respective directions. When activated in the air, similarly, they perform [aerials](https://mcleodgaming.fandom.com/wiki/Aerial_attack) in their respective directions. I recommend their usage since they can make performing attacks easier/faster. AKA “C-Sticks, C Stick, C Sticks”. |
| CSS | “[Character Selection Screen](https://mcleodgaming.fandom.com/wiki/Character_selection_screen)”. |
| CV | Combo video. For example, [this video](https://youtu.be/I_P_LerkzJo). |
| D-Tilt | A down [tilt attack](https://mcleodgaming.fandom.com/wiki/Tilt_attack). |
| Dair | “[Down Air/Aerial”](https://mcleodgaming.fandom.com/wiki/Down_aerial_\(Super_Smash_Flash_2\)). AKA “D-Air”. |
| DC | Disconnect. Usually refers to internet disconnection. e.g. “*Hey you dc’d\!*” |
| DI | “[Directional influence](https://mcleodgaming.fandom.com/wiki/Directional_influence)”. |
| DJC | “[Double Jump Cancel](https://mcleodgaming.fandom.com/wiki/Double_jump_cancel)”. |
| DL | “[Dreamland](https://mcleodgaming.fandom.com/wiki/Dream_Land_\(Super_Smash_Flash_2\))”, a stage. |
| Endlag | “[Ending lag](https://mcleodgaming.fandom.com/wiki/Lag#Ending_lag)”. AKA “Cooldown”. |
| F-Tilt | A forward [tilt attack](https://mcleodgaming.fandom.com/wiki/Tilt_attack). |
| Fair | “[Forward Aerial](https://mcleodgaming.fandom.com/wiki/Forward_aerial)”. AKA “F-Air”. |
| FD | “[Final Destination](https://mcleodgaming.fandom.com/wiki/Final_Destination_\(Super_Smash_Flash_2\))”, a stage. |
| GG | Good game, often said after a match when one still wants to play another. AKA “gg”. |
| GGs | Good games, often said when one has finished a series of matches. AKA “ggs”. |
| KB | “Keyboard” or “[Knockback](https://mcleodgaming.fandom.com/wiki/Knockback)”. |
| KO | “[Knockout](https://mcleodgaming.fandom.com/wiki/KO)”. |
| MDI | “[Multi-directional input](https://mcleodgaming.fandom.com/wiki/Multi-directional_input)”, an advanced technique exclusive to keyboard SSF2. |
| MG | “[McLeodGaming](https://mcleodgaming.fandom.com/wiki/McLeodGaming)”, the developers of SSF2 and other games. |
| Nair | “[Neutral Air/Aerial](https://mcleodgaming.fandom.com/wiki/Neutral_aerial)”. |
| Official | Often refers to the [official McLeod Gaming server](https://discord.com/invite/mcleodgaming).  |
| OoS | “[Out of shield](https://mcleodgaming.fandom.com/wiki/Out_of_shield)” (options). |
| PC | “[Pokémon Colosseum](https://mcleodgaming.fandom.com/wiki/Pok%C3%A9mon_Colosseum)”, a stage, or “Personal Computer”. |
| P2P | “[Peer-to-peer](https://en.wikipedia.org/wiki/Peer-to-peer)”. See the [P2P Section](#3.8-peer-to-peer-\(p2p\)-connection-failed). |
| RAR | “[Reverse aerial rush](https://mcleodgaming.fandom.com/wiki/Reverse_aerial_rush)”. |
| RC Dash | A special technique that gives you a speed boost off platforms. See [this video](https://youtu.be/-ystPgToC-0) for more details. AKA “Arsee Dash”, “Initial Dash Drop”. |
| SD | “[Self-destruct](https://mcleodgaming.fandom.com/wiki/KO#Self-destruct)”. AKA “Suicide”. |
| SH | “[Short hop](https://mcleodgaming.fandom.com/wiki/Short_hopping)”. |
| SSF2 | “[Super Smash Flash 2](https://mcleodgaming.fandom.com/wiki/Super_Smash_Flash_2)”. |
| SV | “[Smashville](https://mcleodgaming.fandom.com/wiki/Smashville)”, a stage. |
| Tech | A [special shield technique](https://mcleodgaming.fandom.com/wiki/Tech). Related to the terms “Teching”, “Tech chase”, etc.  |
| Tilt | “[Tilt attack](https://mcleodgaming.fandom.com/wiki/Tilt_attack)”. AKA “Strong attack”. Comes in three forms: D-Tilt, F-Tilt, U-Tilt. |
| TO | Tournament Organiser. |
| ToS | “[Tower of Salvation](https://mcleodgaming.fandom.com/wiki/Tower_of_Salvation)”, a stage. |
| U-Tilt | An upwards [tilt attack](https://mcleodgaming.fandom.com/wiki/Tilt_attack). |
| Uair | “[Up Air/Aerial](https://mcleodgaming.fandom.com/wiki/Up_aerial)”. AKA “U-Air”. |
| Whiff | When a player misses a move. |
| Whiff punish | Hitting an opponent after they attack you and miss, often while they are stuck in endlag. |
| WOP | “[Wall of pain](https://mcleodgaming.fandom.com/wiki/Wall_of_pain)”. |
| WP | “Well played”, a compliment often said after matches. Sometimes means “Wanna play?”. |
| WR | “[Waiting Room](https://mcleodgaming.fandom.com/wiki/Waiting_Room)”, a stage. |
| YS | “[Yoshi's Story](https://mcleodgaming.fandom.com/wiki/Yoshi%27s_Story)”, a stage. |

For more information on terminology:

* See this video by BananaBoy: [Complete Terminology Guide](https://youtu.be/PMdh6g8Dj98).

* See this page of the Wiki: [Terms \- McLeodGamingWiki](https://mcleodgaming.fandom.com/wiki/Category:Terms).
